# Raw Sass Architecture Structure

## Project Structure

Assuming you named `styles`, you will see the following files generated under a new directory `styles/`:

```scss
styles/
├── abstracts
│   ├── _all.scss
│   ├── _color.scss
│   ├── _functions.scss
│   ├── _keyframes.scss
│   ├── _mixins.scss
│   └── _variables.scss
├── base
│   ├── _all.scss
│   ├── _body.scss
│   ├── _html.scss
│   ├── _normalize.scss
│   ├── _reset.scss
│   └── _custom-reset.scss
├── global.scss
└── index.scss
```
